## Scripts

### Run nodemon
```
$ yarn dev
```

### Build project
```
$ yarn build
```

### Run builded project
```
$ yarn start
```

### Run tests
```
$ yarn test
```

### Collect coverage
```
$ yarn collect-coverage
```

### Build Docker image
```
$ yarn docker:build
```

### Run Docker image
```
$ yarn docker:run
```
###### I recommend using docker extension for this
